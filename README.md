## Description

[Nest](https://github.com/nestjs/nest) framework web app for personal experiments, this repository is meant to run with [bun.sh](https://bun.sh).

## Installation

```bash
$ bun install
```

## Running the app

```bash
# development
$ bun run dev

# production mode
$ bun run start
```

## Test

```bash
# unit tests
$ bun test
```
